# 培训班排房系统

一个用于培训班学员宿舍分配的静态网页系统。

## 功能特点

- 📊 可视化房间状态管理
- 🏨 楼层视图，快速筛选
- 📝 培训班信息管理
- 📄 支持导出分配方案
- 💾 本地数据存储

## 使用方法

1. 直接打开 `index.html` 即可使用
2. 数据自动保存在浏览器本地存储中

## 部署

本系统已部署在 GitHub Pages：
[在线访问](https://你的用户名.github.io/排房系统/)

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- 无需后端服务器
- 使用 LocalStorage 进行数据持久化
